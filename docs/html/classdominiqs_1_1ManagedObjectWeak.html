Detailed Description

Managed Object Weak.

Classes derived from ManagedObjectWeak gracefully support smart pointers (intrusive, strong & weak) Native support for virtual constructors (default & copy) Be carefull not to pass a stack-allocated object to a smart pointer by the way (this is a general rule, not restricted to this class ;-) )
