Detailed Description

Encapsulates a population of chromosomes represented by a vector of doubles.

This software is based on the work of Rodrigo Franco Toso, available at https://github.com/rfrancotoso/brkgaAPI. The most parts of the code are from the original work with minor refactoring.

Original note from rtoso: Encapsulates a population of chromosomes represented by a vector of doubles. We don't decode nor deal with random numbers here; instead, we provide private support methods to set the fitness of a specific chromosome as well as access methods to each allele. Note that the BRKGA class must have access to such methods and thus is a friend.
