Detailed Description

template<class T>
class dominiqs::scoped_ptr_sse2< T >

Simple scoped_ptr to automatically deallocate memory allocated with mallocSSE2 scoped_ptr_sse2 is NOT copyable Users can retake ownership of allocated memory by calling release()
