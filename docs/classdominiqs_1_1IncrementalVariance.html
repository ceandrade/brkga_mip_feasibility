Detailed Description

Unary predicate that incrementally compute the variance of a list of numbers The results can be obtained with result() The flag fromSample decides if the variance was from the entire population or a subset of observation, i.e. if fromSample is true the sum of squares is divided by n - 1, otherwise by n
