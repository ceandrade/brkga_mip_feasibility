Detailed Description

Provides a basic commmand-line application logic.

The purpose of this class is to centralize common stuff such as:

  • Ctrl-C handling
  • parsing command line options / config files
  • open/close global logger
  • exception handling (main try/catch block)

Users must derived from this class to implement custom behaviour. In particular, derived classes MUST implement:

  • checkUsage()
  • exec()

Derived classes MAY also override:

  • readConfig()
  • startup() / shutdown()
