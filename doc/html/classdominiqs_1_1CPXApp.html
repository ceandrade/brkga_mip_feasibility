Detailed Description

Provides a basic Cplex commmand-line application It adds to the basic App the following Cplex-related features:

  • env/problem instances management
  • read common params (time limit, presolve, etc...)
  • read problem from file
  • (optional) presolve Users msut derived from this class to implement custom behaviour.
