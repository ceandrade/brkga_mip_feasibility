66 /* Chapter 4 */
67 typedef struct {int left; int right; double distance;} Node;
68 /*
69  * A Node struct describes a single node in a tree created by hierarchical
70  * clustering. The tree can be represented by an array of n Node structs,
71  * where n is the number of elements minus one. The integers left and right
72  * in each Node struct refer to the two elements or subnodes that are joined
73  * in this node. The original elements are numbered 0..nelements-1, and the
74  * nodes -1..-(nelements-1). For each node, distance contains the distance
75  * between the two subnodes that were joined.
76  */
