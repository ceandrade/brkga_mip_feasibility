Detailed Description

template<typename DataType, typename IndexType>
class ce_andrade::PlainRaggedMatrix< DataType, IndexType >

Plain Ragged Matrix.

Author
Carlos Eduardo de Andrade ce.an.nosp@m.drad.nosp@m.e@gma.nosp@m.il.c.nosp@m.om
Date
2015

This class represents a plain ragged matrix used in the clustering algorithm. This is bottom-left triangular symmetric matrix.

Note
  1. We DO NOT CHECK types or use traits in this template;
  2. DataType must have a simple constructor;
  3. IndexType must be a countable type admitting initialization from zero, pre-increment operator, and less_than operator. Usually, IndexType will be a integer type.
