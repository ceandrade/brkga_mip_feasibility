Detailed Description

template<class Derived>
class dominiqs::ManagedObject< Derived >

Managed Object.

Classes derived from ManagedObject gracefully support intrusive smart pointers. It does not however support weak intrusive smart pointers (

See also
ManagedObjectWeak for that!). Moreover, it does not support non-intrusive pointers. Managed objects are heap only objects! A derived object Derived must be declared as class Derived : private ManagedObject<Derived>...
