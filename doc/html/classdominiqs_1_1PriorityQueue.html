Detailed Description

template<class T, class ID, class ExtractID, class P, class ExtractP, class CompareP>
class dominiqs::PriorityQueue< T, ID, ExtractID, P, ExtractP, CompareP >

Template Class implementing an highly customizable priority queue. It stores objects of type T both by ID and by priority P.

Parameters
Tdata type stored in the container - usually a (smart) pointer to something
IDID type
ExtractIDfunctor type to extract id from objects (key extractor). Must define result_type (=ID)
Ppriority type
ExtractPfunctor type to extract priorities from objects (key extractor). Must define result_type (=P)
ComparePfunctor type to compare priorities
