Detailed Description

ExecutionStopper class.

Author
Carlos Eduardo de Andrade ce.an.nosp@m.drad.nosp@m.e@gma.nosp@m.il.c.nosp@m.om
Date
2015

This class is a singleton used during the algorithm execution to stop by time or user interference. It holds a timer that be controlled from external calls. It also overwrite the Ctrl-C signal handling to allow a softer finish of the algorithm.
